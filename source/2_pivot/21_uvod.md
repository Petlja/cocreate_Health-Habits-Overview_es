# Tablas dinámicas 

Una tabla dinámica es una de las herramientas más importantes para el análisis de datos en programas como *Microsoft Excel* y *Google Sheets*. Permite «dar vuelta» a los mismos datos (pivot = girar) y mostrarlos de una forma nueva: agruparlos, contarlos, calcular promedios o comparar categorías, sin modificar la tabla original. En lugar de contar manualmente y crear varias tablas auxiliares, una tabla dinámica permite obtener respuestas de forma rápida, precisa y clara. 

```{infonote}
En baloncesto, un pivote es un jugador que permanece con un pie en el lugar y, al girar el cuerpo, puede pasar el balón en diferentes direcciones. Del mismo modo, una tabla dinámica utiliza los mismos datos, pero los «gira» y reorganiza para que podamos observarlos desde diferentes ángulos, según lo que queramos averiguar.
```

## Diferencia entre una tabla común y una tabla dinámica

En una tabla común podemos ver cada venta individual, pero si queremos saber, por ejemplo, la cantidad total de fruta vendida de cada tipo, tendríamos que buscar manualmente todas las filas con manzanas y sumar las cantidades, luego las bananas, después las naranjas... Con un número mayor de filas esto lleva tiempo y fácilmente aparecen errores.

**La tabla dinámica organiza y suma los datos automáticamente.** Por ejemplo:

![Tabla dinámica - venta de frutas](images/table3_sr.png)

Este tipo de presentación permite ver los resultados de inmediato, sin sumar manualmente y sin usar fórmulas.

```{infonote}
La tabla común muestra datos individuales. La tabla dinámica muestra su resumen y permite analizarlos.
```
